USER INFORMATION
HTTP Method Path Description
GET /users Gets all users
GET /users/:id Gets a single user by id
POST /users/register Adds a new user
PATCH /users/update/:id Edit a user's information by id
LOGIN /users/login User login
DELETE /users/delete/:id Removes a user by id
ITEM INFORMATION
GET /items Gets all items
GET /items/:id Gets a single item by id
POST /items/addItem Adds a new item
PATCH /items/update/:id Edit an item's information by id
DELETE /items/delete/:id Removes an item by id
ORDER INFORMATION

GET /orders/:id Gets a single order by id
POST /orders/addOrder Adds a new order
PATCH /orders/update/:id Edit an order by id
DELETE /orders/delete/:id Removes a order by id
